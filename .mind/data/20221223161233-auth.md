# auth

| HTTP   | Route         | Description                           |
| ------ | ------------- | ------------------------------------- |
| POST   | /auth/signIn  | login user(with Email/PW or OAuth2)   |
| POST   | /auth/signUp  | register user(with Email/PW or OAut2) |
| GET    | /auth/signOut | sign out user                         |
| DELETE | /auth/:id     | delete user                           |
